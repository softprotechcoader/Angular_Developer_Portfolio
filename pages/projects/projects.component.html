Project Highlights

Revenue Forecast Portal – Cognizant: Internal

Developed a comprehensive revenue forecasting platform utilizing Angular 18 for the frontend and ASP.NET Core Web API for the backend. The application follows a monolithic architecture and implements robust JWT authentication for secure access.
I configured CI/CD pipelines using Azure DevOps to automate deployments, and hosted the app on Azure App Services. The system leverages Azure SQL for efficient data storage and implements a Data First approach with Entity Framework.
Key features include data filtration based on project and year, and revenue forecasts derived from statistical modeling and probability analysis.

Your Shop – E-Commerce App

Designed and developed a simple yet efficient e-commerce application using Angular, featuring full CRUD operations for managing products. The platform supports product addition, modification, and deletion, alongside a smooth user interface designed with Bootstrap.
Key features include an intuitive shopping cart, seamless product management, and integrated payment services. The application ensures high performance with optimized backend APIs for fast response times.

Seat Booking App

Developed a full-featured seat booking system for restaurants, incorporating user registration, seat locator, restaurant locator, menu viewing, and payment processing.
The app facilitates seamless seat reservations, allowing customers to view available slots in real-time. It also integrates restaurant location data and displays the menu, offering a streamlined user experience from booking to payment.

Task Tracker – Web & Windows App

Built a versatile task tracking application for both web and Windows platforms. The app supports task assignment with calendar integration, project duration tracking, and automated delay notifications.
I converted the web version into a Windows application using ElectronBuilder, ensuring cross-platform accessibility. Data is stored in a JSON file, and users can export task data to Excel for detailed reporting.
